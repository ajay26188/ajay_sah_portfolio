---

## Key Features

- **Random Anecdotes**: Display random programming anecdotes with each click.  
- **Voting System**: Vote for your favorite anecdotes in real time.  
- **Top Anecdote**: Always shows the most-voted anecdote.  
- **Leaderboard**: Dedicated page listing all anecdotes with their vote counts.  
- **Persistence**: Votes are stored in `localStorage`, keeping them across sessions.  
- **Smooth Animations**: Transitions and voting feedback powered by Framer Motion.  
- **Responsive Design**: Custom CSS for styling and mobile-friendly layouts.

## Technologies Used

- **Frontend**: React (with Vite)  
- **Routing**: React Router DOM  
- **State Management**: React Hooks (useState, useEffect)  
- **Persistence**: localStorage API  
- **Animations**: Framer Motion  
- **Styling**: CSS3 with responsive design  
- **Deployment**: Netlify

## Challenges & Learnings

- Implementing a **random anecdote generator** while ensuring smooth user experience.  
- Handling **vote state persistence** between browser sessions using localStorage.  
- Designing engaging **animations and transitions** with Framer Motion.  
- Structuring multiple pages with **React Router DOM** for seamless navigation.
