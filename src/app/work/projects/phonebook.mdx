## Overview

**Phonebook** is a simple yet practical contact management app built with React.  
The project demonstrates key concepts in frontend development such as **state management, controlled forms, client-side routing, and local persistence with localStorage**.  

The app allows users to **add, search, update, and delete contacts**, making it a functional personal tool and a strong portfolio project showcasing CRUD operations in React.

Live Demo: [Phonebook Website](https://ajaysphonebook.netlify.app/)  

---

## Key Features

- **Add Contacts**: Easily create new contacts with name & phone number.  
- **Search & Filter**: Real-time filtering of contacts by name.  
- **Update & Delete**: Modify existing contacts or remove them completely.  
- **Persistence**: Contacts are stored in `localStorage`, ensuring data is saved across sessions.  
- **Routing**: Basic navigation implemented with React Router for a smooth user experience.  

---

## Technologies Used

- **Frontend**: React + JavaScript  
- **Routing**: React Router  
- **State Management**: React Hooks (useState, useEffect)  
- **Storage**: localStorage API for data persistence  
- **Styling**: CSS   

---

## Challenges & Learnings

- Building a **CRUD workflow** with React while keeping state updates predictable.  
- Managing **local persistence** with localStorage and syncing it with React state.  
- Gaining experience with **React Router** for client-side navigation.  
- Writing cleaner, reusable components for form handling and lists.  

---

## Outcome

The Phonebook app demonstrates:  
- A strong grasp of React fundamentals (hooks, controlled forms, lists).  
- Practical implementation of **localStorage persistence**.  
- A user-friendly, single-page contact manager.  

This project serves as both a **hands-on learning project** and a **solid portfolio piece** highlighting frontend CRUD app development.
